============
SERVER FILES
============
The Server folder has all of the classes for the Daemon and server.
Execute the ConnectFourDaemon class on the server and it will listen 
at port 3045.


============
CLIENT FILES
============
The Client folder has all of the classes and files for the client program.
Execute the ConnectFourClient class and it will try to connect to the specified server on port 3045.
$java ConnectFourClient <SERVER_NAME_OR_IP>


============
TO PLAY
============
Requires 2 players.  For the client, copy the client to individual machines and specify the game server on the command line as above.
